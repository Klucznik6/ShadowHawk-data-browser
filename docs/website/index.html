Powerful Features

Everything you need for professional database browsing and data analysis

Ultra-Fast Performance

Powered by Polars integration for 5-30x faster data processing. Multi-threaded operations ensure responsive UI even with large datasets.

  • Smart caching system
  • Memory-efficient processing
  • Real-time progress indicators

Professional Interface

Clean tkinter-based GUI with intuitive tabbed layout, context menus, and keyboard shortcuts designed for efficient database browsing.

  • Context menus & shortcuts
  • Status bar with feedback
  • Customizable font sizes

Advanced Search Engine

Global search across all loaded databases with ultra-fast parallel processing. Interactive results with click-to-navigate functionality.

  • Real-time filtering
  • Cross-database search
  • Search highlighting

Multi-Format Support

Open SQLite databases, CSV files, Excel spreadsheets, JSON data, and Access databases. Intelligent auto-detection and encoding support.

  • SQLite (.db, .sqlite, .sqlite3)
  • CSV with auto-detection
  • Excel (.xlsx, .xls)
  • JSON with nested support
  • Access (.mdb, .accdb)

Smart Database Management

Persistent sessions remember all databases between restarts. Recent files menu and automatic restoration of your last working session.

  • Auto-save functionality
  • Session restoration
  • Database removal controls

Professional Analysis Tools

SQL query interface, comprehensive statistics, data export capabilities, and column analysis with min/max/mean calculations.

  • SQL query execution
  • Export to CSV/Excel/JSON
  • Column statistics
